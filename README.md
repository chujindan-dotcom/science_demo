# Science Demo · 科普演示合集

用交互式 H5 动画做科学科普。所有演示均为纯前端静态页面,无需构建、无需安装,任何现代浏览器(桌面 / 手机 / 平板)打开即用。

## 演示列表

### 🪐 宇宙天体 3D 演示([solar-system/](solar-system/))

太阳系交互式 3D 演示,基于 Three.js(WebGL):

- **太阳系天体**:太阳(动态火焰着色器 + 日冕泛光)、八大行星、月球、土星环(含卡西尼缝)、小行星带
- **深空天体(天球背景层)**:
  - **二十八宿**:完整 28 宿,按东方青龙 / 北方玄武 / 西方白虎 / 南方朱雀四象分色,连线成图、标注中文宿名,采用近似真实赤道坐标
  - **银河**:沿真实银道面(银北极 RA 12h51m / Dec +27°)分布的乳白色恒星带
  - **M42 猎户座大星云**:位于参宿,红 / 青双色辉光 + 年轻星团
- **银河系俯瞰模式**:一键把镜头拉到银盘外,俯瞰整个棒旋星系;太阳系被放在盘面约一半半径的旋臂上(真实约 2.6 万 / 5 万光年),银心朝人马座方向
- **运动**:按真实周期比例的公转与自转(含金星 / 天王星的逆行自转、天王星 98° 的"躺姿"、月球潮汐锁定)
- **材质**:全部程序化生成——岩石陨石坑、地球海陆云层、气态巨行星条纹与大红斑、冰巨星甲烷蓝,不依赖任何外部贴图
- **交互**:拖拽旋转视角、滚轮 / 双指缩放、时间流速滑块(静止 ~ 1 秒 120 天)、点击天体 / 星宿聚焦并显示中文科普信息、轨道 / 标签 / 星宿开关

> 尺度说明:银河系、M42、二十八宿相对太阳系是"无穷远的背景方向",只贴在天球层上、不参与太阳系缩放,也不随时间滑块运动(周日 / 周年视运动属于地球,日心场景中忽略)。天球层按黄赤交角 23.44° 正确倾斜。

> 注:天体大小与轨道距离经过压缩,并非真实比例(真实比例下行星在屏幕上只有不到一个像素)。公转与自转周期之间的相对比例是真实的。

## 本地运行

推荐用附带脚本启动,它会**绑定所有网卡(`0.0.0.0`)**并打印局域网访问地址,方便手机/平板真机测试:

```bash
./serve.sh          # 默认端口 8123,绑定 0.0.0.0
./serve.sh 8080     # 指定端口
```

或手动启动(同样绑定所有 IP):

```bash
python3 -m http.server 8123 --bind 0.0.0.0
# 或
npx serve -l tcp://0.0.0.0:8123
```

- 本机打开:http://localhost:8123/
- 局域网真机打开:http://<本机局域网IP>:8123/(手机需连同一 WiFi;若打不开,检查防火墙是否放行该端口)

页面需联网加载 Three.js CDN(unpkg)。也可直接访问已部署的在线版:<https://chujindan-dotcom.github.io/science_demo/>

## 技术栈

- [Three.js](https://threejs.org/) r160(ES Modules + import map,无构建步骤)
- 原生 HTML / CSS / JavaScript
- 纹理全部由 Canvas + 值噪声(value noise / fBm)程序化生成
